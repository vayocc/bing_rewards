## 前置准备

安装node 后检测版本要大于18
```shell
node -v
```
 安装依赖 
```shell
npm install 
```
或者从镜像安装
```shell
npm install --registry=https://registry.npmmirror.com
```

安装playwright浏览器
```shell
npx playwright install
```

启动方式
pc端
```shell
multi_rewards.js 
```
or
```shell
multi_rewards.js --pc
```
手机端
```shell
multi_rewards.js --mobile
```
